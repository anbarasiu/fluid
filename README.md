## SEO Enhancements

This blog includes comprehensive SEO optimization:

- **Meta Tags**: Enhanced title, description, and keywords
- **Open Graph**: Rich previews on social media
- **Twitter Cards**: Beautiful link previews
- **JSON-LD Structured Data**: Blog and Person schemas
- **jekyll-seo-tag**: Automated SEO optimization
- **AI Crawler Permissions**: GPTBot, Claude-Web, PerplexityBot, etc.
- **Sitemap**: Auto-generated by Jekyll
- **RSS Feed**: For subscribers

## Local Development

```bash
# Clone the repository
git clone https://github.com/anbarasiu/fluid.git
cd fluid

# Install dependencies
bundle install

# Run locally
bundle exec jekyll serve

# Open browser
open http://localhost:4000
```

## Writing Posts

Create a new file in `_posts/` with the format: `YYYY-MM-DD-title.md`

```markdown
---
layout: post
title: "Your Post Title"
description: "Brief description for SEO (100-160 chars)"
tags: [software-engineering, web-development]
---

Your content here...
```
